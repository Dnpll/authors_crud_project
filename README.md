# 🧑‍💻 Author CRUD API

Prosty projekt demonstracyjny stworzony w Python (FastAPI + SQLite), pokazujący pełny CRUD (Create, Read, Update, Delete) dla encji **Author**.

---

## ⚙️ Technologie

- **Backend:** FastAPI, SQLAlchemy, Pydantic  
- **Baza danych:** SQLite  
- **Frontend:** prosty HTML + Fetch API  
- **API:** REST
